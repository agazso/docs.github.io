      Installation

      Supported platforms

      The client can be installed on MacOS and various Linux flavors.

      Installing the Bee binary

      At every official release, we release the binary files of our software. You can download them by navigating to releases.

      Installation from source

      Prerequisites for installing Bee from source are:

      • Go - download the latest release from golang.org,
      • git - download from git-scm.com or install with your package manager,
      • make.
      1. Clone the repository:
      git clone https://github.com/ethersphere/bee
cd bee
      1. Use git to find the latest release:
      git describe --tags
      1. Checkout the latest release:
      git checkout <name_of_latest_tag>
      1. Build the binary and move the binary to your path:
      make binary
cp dist/bee /usr/local/bin/bee
      1. You should now be able to run the bee command, which you can verify by running:
      bee version

      Which should return the version number of the Bee you are running.
